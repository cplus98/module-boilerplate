module-boilerplate

How to make

1. Clone this project.
2. Change project names.
3. Add/Remove required modules in "package.json".
4. Make your own codes.
5. Build library. ($ yarn build)
6. Upload to Git.


Is this module for React-Native? Change devDependencies in "package.json" file.

1. Remove "@types/react-dom"
2. Add "@types/react-native": "^0.61.0"
